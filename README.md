# 隨心使魔

## Build Setup

``` bash
# install dependencies
$ npm install

# server testing
$ npm run start

```

### 初步使用Git Bash連結Heroku
```bash
$ heroku login # 先登入Heroku的帳密
```


### Clone The Bot From Heroku Git
```bash
$ git clone https://git.heroku.com/tgfollowheartbot.git
```

### 檢視當前Heroku Log狀態
```bash
$  heroku logs -t
```
